# Writing Samples

Drop 1–3 samples of your own professional writing here (a past cover letter, a LinkedIn About section, any prose you have written) — plain `.md` or `.txt` files.

The system reads these to calibrate generated cover letters, outreach, and form answers to *your* voice. It extracts only abstract style descriptors (tone, sentence length, vocabulary) — never your content, claims, or personal identifiers.

This README is system-owned documentation; everything else you add here is yours and is gitignored by default.
